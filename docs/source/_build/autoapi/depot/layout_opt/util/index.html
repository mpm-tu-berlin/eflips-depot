class depot.layout_opt.util.StoppingCriteria¶

Stopping utilities for optimization loops of an evolutionary algorithm.

select(*args)¶

Select the stopping criteria. Pass at least one stopping criteria method name as str. The same method cannot be selected twice.

check(*args, **kwargs)¶

Return True if any of the selected stopping criteria returns True. Pass parameters as kwargs based on stopping criteria requirements.
